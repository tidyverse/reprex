# Venue-specific shortcuts

These are thin wrappers around
[`reprex()`](https://reprex.tidyverse.org/dev/reference/reprex.md) that
incorporate the target `venue` as a suffix in the function name, for
easier access via auto-completion.

## Usage

``` r
reprex_html(...)

reprex_r(...)

reprex_rtf(...)

reprex_slack(...)
```

## Arguments

- ...:

  Passed along to
  [`reprex()`](https://reprex.tidyverse.org/dev/reference/reprex.md).
